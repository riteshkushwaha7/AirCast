# Product UX Notes

## Design Intent
- Calm, premium minimal interface
- Whitespace-first layout
- Soft neutral palette
- AQI colors only where meaningful
- One chart on dashboard

## Dashboard Information Density
The dashboard intentionally keeps only:
- greeting + location
- current AQI + category
- 4h / 12h / 24h forecast chips
- one recommendation sentence
- one compact trend chart
- best outdoor window
- notification status

## Mobile-First Decisions
- Bottom navigation for core app routes on small screens
- Sidebar navigation only on medium+ screens
- Cards are single-column first, then split into compact grid

## Language Tone
Recommendations and alerts use short, calm wording:
- "Wear a mask and avoid long outdoor exposure."
- "Sensitive users should reduce exposure."
- "Best outdoor window: 7:00 AM - 8:30 AM"
